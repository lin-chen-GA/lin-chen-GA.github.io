Linux Commands
ls //list the content in the current directory

pwd //show the current directory

mkdir folderName //create a folder

cd folderPath //enter the folder

cd .. //back to upper layer folder

cd //back to the home directory

rm fileNmae //remove a file

rm -rf folderName //remove a folder

mv oldFileName newFileName //rename a file

tab key //use the tab key to complete typing faster
Vim Editor
Control mode //control operations, default mode
Edit mode //be able to type in characters

from Control mode to Edit mode, 'i' or 'a'
from Edit mode to Control mode, Escape

:w //save
:wq //save and quit vim
:q //quit vim
:q! //force quit, may lose some information

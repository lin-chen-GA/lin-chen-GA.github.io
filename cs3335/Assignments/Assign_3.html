Assignment 3: CSV to HTML
Description
  • Write a C program to meet the following requirements
    1. Convert a csv file to a html file. The html file should have a table containing the content of the csv file
    2. Read the input csv file from command line
    3. The output file name should be "index.html"
    4. Code Layout
      • Save *.h files in the "include" folder
      • Save *.c files in the "src" folder
    5. Makefile
    6. Necessary comments
    • Hints
  • Html
  • Table in HTML
    • Test Data
  • Input: input.csv
  • Output: index.html
    • Grading Criteria
  • The program that is not able to be compiled with makefile will receive a zero
  • The program that has no the required code layout will receive a zero
  • The program can read the input file name from the command line (20%)
  • The program can read the input csv file (30%)
  • The program can output index.html correctly (30%)
  • The program can be compiled and cleaned with make(10%)
  • No warning information from compiling (10%)
    • Submission
  • Compress the folder containing the code to "Assign_3.tar"
  • Submit the compressed tar file to the BlazeView

Create Project
  1. Create a folder for the program, such as "Program_1"
  2. Enter the created folder
  3. Create a C file, such as "hello.c"
  4. Write the code and save
  5. Compile the code with "gcc hello.c"
  6. Run the executable program, "./a.out"
Knowledge Points
  1. C language has the concept of class which we learned in Java
  2. Comments
    • /* ... */, block comment
    • //, line comment
  3. {}, delimit the body of a function
  4. main() function is the starting point of a C program
  5. ;, end of a line
  6. C is case sensitive
  7. main() function need a returned integer, by default, 0 means the program has been executed successfully, non-zero numbers represent the non-successful execution
  8. Reserved words, or keywords, have a specific meaning to the compiler and cannot be used as the variable names
  9. a.out is the default output file by gcc complier. To have a specific output file name, use "gcc -o output_file_name source_code.c"

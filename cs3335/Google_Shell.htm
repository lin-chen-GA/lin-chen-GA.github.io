Google Cloud Shell
Basic Operations
  1. Access Google Cloud
  2. Start Shell
    • Click "Select a Project"

    • Click "New Project"

    • Add a name to the project

    • Wait a while for creating the virtual machine

  3. Edit Files
    • Editor
    • vim
  4. Upload Files
  5. Download Files
    • Download Tool, download a single file
    • cloudshell download test1.md test2.md, download multiple files
  6. Install Softwares
    • sudo apt-get install [package_name]

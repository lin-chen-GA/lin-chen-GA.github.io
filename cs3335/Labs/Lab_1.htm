Lab 1: Linux and Vim
  1. Under your home directory (/home/your_user_name), create "Lab_1" folder, do not use special characters and space in the folder name
  2. Enter "Lab_1" folder
  3. Use Vim to create and edit "Hello.java"
    4. public class Hello
{
	public static void main(String args[])
	{
		System.out.println("Hello World!");
	}
}
  4. Save the file and Quit Vim
  5. Use "javac Hello.java" to compile the code
  6. Use "java Hello" to run the code
  7. Go back to the home directory (/home/your_user_name)
  8. Compress the "Lab_1" folder by "tar -cvf Lab_1.tar Lab_1"

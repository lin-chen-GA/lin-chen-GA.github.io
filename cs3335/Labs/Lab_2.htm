  • Create local system folder
    1. Under your home directory (/home/username), create a folder ".local"
    2. Under the ".local" folder you created, create a folder "bin"
    3. Copy "info" to your "./local/bin" folder
  • Add the local system folder to search path
    1. Under your home directory, "vim .bashrc"
    2. Add "export PATH="$HOME/.local/bin:$PATH"" at the end of the .bashrc file
    3. Save and Quit .bashrc file
    4. Under your home directory, "source .bashrc"

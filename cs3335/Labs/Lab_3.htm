Lab 3: Random Number
1. Learn rand()
  • rand() function generates a pseudo-random integral number in the range between 0 and RAND_MAX
  • rand() is defined in stdlib.h
  • Run the code below a couple of times, you can see the same random sequence
    • #include <stdio.h>
#include <stdlib.h>

int main()
{
	for(int i = 0; i < 10; i++)
		printf("%d\n", rand());

	return 0;
}
  • Use the system time as the seed
  • srand() is the fuction to take a seed for generating random numbers, which is defined in stdlib
  • time(NULL) take the system's current time, which is defined in time.h
  • Run the code below a couple of time, observe the random numbers generated
    • #include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main()
{
	srand(time(NULL));

	for(int i = 0; i < 10; i++)
		printf("%d\n", rand());

	return 0;
}

First Java Code
/* Welcome.java
 * CS 1301
 */

public class Welcome
{//start brace

	// main function, starting point of running
	// String args[], parameters
	public static void main(String args[])
	{
		// print out message to standard output
		System.out.println("Welcome to Java!");// semiicolon
	}

}//end brace
Create Project
  1. Create a folder for the project
  2. Enter the created folder
  3. Create a Java file
  4. Write the code
  5. Save it, make sure the file name is same as the class name
  6. Compile and run the code
Knowledge Points
  1. The class name should be same as the file name, and starts with an uppercase letter by convention
  2. Comments
    • /* ... */, block comment
    • //, line comment
  3. {}, delimit the body of a class or method
  4. System.out.println, display a line of string
  5. "...", a string
  6. ;, end of a line
  7. Java is case sensitive
  8. Reserved words, or keywords, have a specific meaning to the compiler and cannot be used as the variable names

Mathematical Computations
public class ComputeExpression{
	public static void main(String[] args)
	{
		System.out.print("(10.5+2*3) / (45 - 3.5) = "); // print without changing to a new line
		System.out.println((10.5+2*3) / (45 - 3.5)); // println, move to the beginning of the next line
	}
}
Programming Style
  • Indentation
    • Illustrate the structural relationships between a program's components or statements, at least two spaces
  • Spacing
    • A single space should be added on both side of a binary operator
  • Block Styles 
      //next-line style
public class Welcome

		System.out.println("Welcome to Java!");
	}

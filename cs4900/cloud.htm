Linux Commands

mkdir folderName //create a folder

cd folderPath //enter the folder

cd .. //back to upper layer folder

cd //back to the home directory

rm fileNmae //remove a file

mv oldFileName newFileName //rename a file

tab key //use the tab key to complete typing faster
Vim Editor
Control mode //control operations, default mode
Edit mode //be able to type in characters

from Control mode to Edit mode, 'i' or 'a'
from Edit mode to Control mode, Escape

:w //save
:wq //save and quit vim
:q //quit vim
:q! //force quit, may lose some information

    Exercise 3: Rename files
    1. Create a shell script "rename.sh", copy the following script code to it
    2. Type "chmod 755 rename.sh" to change the "rename.sh" to be a executable file
      3. #!/bin/bash
 
files=`find $1 *.*`
 
for f in $files
do
	if [ ${f:(-3)} = "txt" ]
	then
		mv $f ${f/%.txt/.dat}
	fi
done
    3. Create a folder "Exercise_3"
    4. Type "touch Exercise_3/f1.txt Exercise_3/f2.txt Exercise_3/f3.txt" to create a few empty files in the folder "Exercise_3"
    5. Type "./rename.sh Exercise_3" to rename .txt files to .dat files

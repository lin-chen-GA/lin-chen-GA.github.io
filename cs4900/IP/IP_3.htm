IP 3: Generating Documents
Project Architecture

1. Add Exception
  • Modify your code for IP-2
  • In primeio.py, raise the necessary Exceptions from write_primes and read_primes
  • In primemodule.py, raise appropriate Exceptions when n is not greater than 0 in isPrime
  • Use try ... except ... to call the functions and handle exceptions in generate.py
    • 2. Docstring
  • Update docstring in each module
    • Each module has the module docstring
    • The docstring for each function should contains function description, args (if have parameters), returns (if have returns), exceptions (if have exceptions), examples
    • 3. Generate Documents
  • Generate documents with sphinx, and save the document to /doc folder

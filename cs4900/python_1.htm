Create Install Fold (if not exist)
  • mkdir .local in your home directory, such as /home/lchen
  • mkdir bin in .local
  • mkdir lib in .local
  • Edit .profile, "export PATH="/opt/anaconda/bin:$HOME/.local/bin:$PATH"
  • Edit .profile, "export PYTHONPATH="$HOME/.local/lib:$PYTHONPATH"
  • source .profile
Create a Program
# hello.py
#!/usr/bin/python

def main():
    print("Hello World!")

if __name__ == '__main__':
    main()
Publish Executable Program
chmod 755 hello.py
cp hello.py hello
mv hello ~/.local/bin

hello
Publish Library
mv hello.py ~/.local/lib
